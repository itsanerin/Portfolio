DOUGH RE MI

In November, 2019, I developed a demo website for a hypothetical bakery, Dough Re Mi, in order to showcase my skills in Ruby with the New York Code and Design Academy. This is my first website built using a back-end language like Ruby to manage data.

The project required that I utilize object-oriented programming (OOP) in order to show a bakery's product information. There are different types of products and an image, title, price, and description for each one.
